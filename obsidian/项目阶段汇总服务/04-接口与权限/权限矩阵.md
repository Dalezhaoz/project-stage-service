| --- | --- | --- | --- |
| 查看全部项目 | 是 | 是 | 否 |
| 查看自己项目 | 是 | 是 | 是 |
| 管理用户 | 是 | 否 | 否 |
| 管理服务器 | 是 | 是 | 否 |
| 测试 Agent | 是 | 是 | 否 |
| 配置中心库 | 是 | 否 | 否 |
| 配置调度 | 是 | 否 | 否 |
| 配置负责人 | 是 | 是 | 否 |
| 配置应用服务器分配 | 是 | 是 | 仅自己项目 |
| 手工刷新 | 是 | 是 | 取决于 allowUserRefresh |
