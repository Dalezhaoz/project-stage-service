# 查询接口

## `/api/test`

用途：

- 测试服务器是否能通过 Agent 成功访问源库

当前实现：

- 主服务不直连数据库
- 主服务调用 agent 的 `/test`

## `/api/refresh`

用途：

- 触发一次完整刷新

当前实现：

- 调用各 Agent 的 `/query`
- 聚合后由主服务统一写中心表
- 再回写本地 SQLite

## `/api/query`

用途：

- 返回看板汇总结果

当前实现：

- 只查中心表
- 对 external 用户做服务端过滤

## `/api/stages`

用途：

- 返回当前条件下可见的阶段名列表

当前实现：

- 从中心表记录中提取

## `/api/export`

用途：

- 导出 Excel

当前实现：

- 从中心表读取并导出
