# 发布检查清单

## 主服务

- 服务器配置是否包含 `agentPort`
- 服务器配置是否包含 `agentSecret`
- 中心库是否已启用
- 中心库测试连接是否成功
- 调度时间是否正确

## Debian Agent

- `stage_agent.py` 是否更新到最新版本
- `cryptography` 是否已安装
- `STAGE_AGENT_SECRET` 是否已配置
- `systemd` 服务是否启动
- `/health` 是否正常

## Windows Agent

- 程序是否更新
- `AgentSecret` 是否已配置
- Windows Service 是否启动
- `/health` 是否正常

## 联调

- `/api/test` 是否成功
- `/api/refresh` 是否成功
- 中心表是否新增 `synced_at`
- 页面查询是否能返回数据
