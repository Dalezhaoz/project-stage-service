# 数据流转

## 刷新链路

1. 主服务读取已启用服务器
2. 主服务为每台服务器构造 `QueryDefinition`
3. 主服务使用 Agent 密钥加密请求
4. Agent 解密请求
5. Agent 在本机查询业务库
6. Agent 返回标准化记录
7. 主服务映射成 `ProjectStageRecord`
8. 主服务构建 `ProjectStageSummary`
9. 主服务写中心表
10. 主服务回写本地 SQLite 缓存

## 查询链路

1. 前端提交筛选条件
2. 主服务从中心表读取数据
3. 服务端执行权限过滤
4. 返回看板数据、分组和统计摘要

## 配置链路

1. 管理员维护服务器配置
2. 配置中保存：
   - 源库主机、端口、账号密码
   - Agent 端口
   - Agent 密钥
3. 服务器配置保存在本地加密文件
