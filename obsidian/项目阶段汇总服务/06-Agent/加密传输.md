# 加密传输

## 当前方案

- 主服务和 Agent 之间使用共享密钥
- 主服务请求体在发送前使用 AES-GCM 加密
- Agent 收到请求后解密

## 当前保护对象

- 源库账号
- 源库密码
- 中心库账号
- 中心库密码
- 查询定义内容

## 当前主服务配置位置

- 服务器配置里的 `AgentSecret`

## 当前 Agent 配置位置

### Debian

- 环境变量 `STAGE_AGENT_SECRET`

### Windows

- `appsettings.json` 中的 `AgentSecret`

## 当前建议

- 每台 Agent 使用独立密钥
- 密钥长度尽量随机且足够长
